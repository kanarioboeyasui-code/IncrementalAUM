# Investment Incremental

**Investment Incremental** は、クオンタムファンドのマネージャーとなり、データ収集、市場分析、投資戦略の開発を通じて運用資産（AUM）を増やしていくインクリメンタルゲーム（放置ゲーム）です。

## 🎮 ゲームの概要

プレイヤーはインターンとしてキャリアをスタートし、データ収集から始めます。集めたデータを分析して「研究ポイント」を獲得し、市場の「ファクター（要因）」を発見していきます。新しいファクターを発見することで、より高度な投資戦略（インデックスファンド、モメンタムファンド、ヘッジファンドなど）を開発できるようになります。

## ✨ 主な機能

- **データ収集 (Data Collection)**: インターンやWebスクレイパー、衛星フィードなどを駆使して、市場の生データを収集します。
- **リサーチ & 分析 (Research & Analysis)**: アナリストやMLクラスター、量子ラボを用いてデータを分析し、研究ポイントに変換します。
- **ファクター発見 (Factor Discovery)**: 研究ポイントを消費して、市場を動かす隠れたファクター（Beta, Size, Value, Momentum, AlphaOmegaなど）を解明します。
- **投資戦略 (Investment Strategies)**: 発見したファクターに基づいて、新しい投資ファンドを立ち上げます。戦略を多様化（分散投資）することで、市場のリスクに備えることができます。
- **マーケットイベント (Market Events)**: 市場は常に変動します。暴落（Crash）や高騰（Fever）が発生した際、分散投資が効いていると資産を守り、あるいは大きく増やすチャンスとなります。
- **リバース (Reverse / Prestige)**: すべてのファクターを解明すると、新たな地域（Region）へ進出できます。資産やアップグレードはリセットされますが、強力なボーナスを得て再スタートできます。

## 🕹️ 遊び方

1. **データを集める**: 「データ」タブでインターンなどを雇い、データポイントを自動生成します。
2. **研究を進める**: 「リサーチ」タブでアナリストなどを雇い、データポイントを消費して研究ポイントを生成します。
3. **ファクターを発見する**: 研究ポイントが一定量貯まると、自動的に新しいファクターが発見されます。
4. **戦略を開発する**: 「戦略」タブで、発見したファクターに対応するファンドを開発します。ファンドは自動的にお金を稼ぎます。
5. **資産を増やす**: 稼いだお金でさらに高度な設備や人材を雇い、成長を加速させます。
6. **分散投資**: 複数の異なる戦略を持つことで「分散度」が高まり、市場暴落時のダメージを軽減できます。
7. **リバース**: ゲームをクリア（AlphaOmegaファクター発見）すると、次の周回プレイへ進めます。

## 🚀 インストールと実行方法

このゲームは単一のHTMLファイルで動作します。特別なインストールは不要です。

1. リポジトリをダウンロードまたはクローンします。
2. `index.html` ファイルをブラウザ（Chrome, Firefox, Edgeなど）で開きます。
3. すぐにゲームが始まります！
